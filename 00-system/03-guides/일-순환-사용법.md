# 일 순환 사용법 — 매일 이렇게 씁니다

> 클로드코드를 처음 쓰는 분 기준으로 적었습니다. 명령어를 외울 필요 없습니다 — 전부 **한국어로 말하면** 됩니다.
> (클로드코드 실행 자체가 처음이면 [[claude-code-practice-guide]] 먼저 — 터미널에서 `claude` 한 단어로 시작합니다.)

## 이 워크스페이스가 매일 하는 일 (그림 하나)

```
흩어진 채널(문자·카톡·메일·붙여넣기)
   → ① 모은다   (수집기가 00-inbox/raw/에 원본 그대로 쌓음)
   → ② 가른다   (AI가 "이건 일정·할일·회신·참고·버림" 판정 — 승인은 당신)
   → ③ 제자리에 (할일 파일·일기·진행표 같은 정해진 파일로)
   → ④ 돌아본다 (아침 상태판 — 뭐가 밀렸고 뭐가 이상한지)
   → ⑤ 한다 / ⑥ 남긴다 (일하고, 배운 것은 위키에)
```

당신이 외울 것은 아래 **말 몇 마디**가 전부입니다.

---

## 아침 — "morning" 이라고 치세요

하루의 시작. 클로드에게 `morning` 또는 "좋은 아침"이라고 치면 상태판이 나옵니다:

```
Morning Routine - 2026-07-08 (수)

[done] 1. 동기화        -- 원격과 같음
[todo] 2. 카톡 수집     -- 마지막 체크 26시간 전 → 이어서 바로 돌립니다
[done] 3. 문자 수집     -- 오늘 새벽 정상 실행
[todo] 4. 미처리 수집분 -- 2건 → 분류 이어감
[info] 5. 오늘 할 일    -- Today 3건
```

읽는 법은 세 가지만 알면 됩니다:
- **[done]** — 정상. 아무것도 안 해도 됨.
- **[todo]** — 시스템이 이어서 처리하거나, 당신 승인이 필요한 것. 가만히 있으면 클로드가 순서대로 이어갑니다.
- **[warn]** — 점검 자체가 실패한 것. "이거 왜 그래?"라고 물어보면 됩니다.

상태판 뒤에 클로드가 알아서 이어갑니다: 오늘 일기(데일리 노트)를 만들고 → 밀린 수집을 돌리고 → 쌓인 수집분의 분류를 **제안**합니다. 당신은 제안 표를 보고 "응" 또는 "2번은 빼고"라고 답하면 끝.

## 수집 — 대부분 자동, 나머지는 붙여넣기

- **자동**: 연결해 둔 채널(문자·카톡 등)은 몇 시간마다 알아서 쌓입니다. 성공하면 조용하고, **고장 났을 때만** 알림이 옵니다.
- **수시로 당기고 싶으면**: "수집 돌려줘" 한 마디.
- **연결 안 된 채널**(인스타 DM, 종이 문서 사진 등)은 그냥 대화에 붙여넣으세요: "이 DM 처리해줘" + 내용 붙여넣기. 그게 곧 수집입니다.

**원칙 하나**: 모인 원본은 지우지도 고치지도 않습니다. 처리가 끝나면 "처리됨" 표시만 붙습니다. 나중에 언제든 되짚을 수 있습니다.

## 가르기 — AI가 판정하고, 당신은 승인만

수집분이 쌓이면 "인박스 분류해줘"라고 하거나, 아침 흐름에서 자동으로 이어집니다. 이런 표가 나옵니다:

```
| # | 내용 요약                  | 판정      | 하려는 것            |
|---|---------------------------|-----------|---------------------|
| 1 | 거래처 "다음 주 미팅 가능?" | 회신 필요  | 회신 초안 준비       |
| 2 | 카드 승인 12,000원         | 노이즈     | 태그 붙여 보관       |
| 3 | "금요일까지 자료 제출"      | 할일       | 할일 목록에 추가     |
```

여기서 중요한 규칙 — **일의 수위가 세 가지**로 갈립니다:

1. **되돌릴 수 있는 정리** (보관·태그·할일 추가) → AI가 바로 하고 결과만 보고합니다. 하나하나 물어보지 않습니다.
2. **큰 변경** (진행표의 거래 상태 바꾸기 등) → 바뀐 모습을 먼저 보여주고 승인을 기다립니다.
3. **밖으로 나가는 것** (답장 발송, 일정 등록, 돈) → AI가 **초안까지 전부** 만들어 놓고, 마지막 "보내줘"만 당신이 합니다. 이건 예외가 없습니다.

## 회신 — 초안은 AI가, 발송은 당신이

"회신 써줘" + 상대 원문을 주면(가르기에서 이어지면 자동), 클로드가 **회신 전문**을 만들어 보여줍니다. 보내기 전에 스스로 검사도 합니다 — 당신이 정한 금지 표현(예: 본문에 가격 쓰지 않기)이 들어갔으면 스스로 고칩니다.

마음에 들면 "보내줘". 고치고 싶으면 "도입부가 길어" 같은 말로 — 부분 수정이 아니라 처음부터 다시 검증하며 새 초안을 만듭니다.

## 장부 맞추기 — 표 데이터는 다른 문으로

은행 거래내역 같은 **표 파일**은 붙여넣기가 아니라 정해진 폴더에 내려받아 두고 "장부 맞춰줘"라고 하면 됩니다. 발행한 것과 입금된 것을 대조해 "받을 돈이 며칠째 안 들어왔는지"를 장부 파일에 정리합니다. 여기서도 "입금 확인" 할일을 추가할지는 당신이 결정합니다.

## 빈칸 채우기 — 쓸수록 똑똑해지는 지점

이 워크스페이스의 개인화는 코드가 아니라 **파일 두 개의 빈칸**입니다 (`00-system/08-registry/`):

- **자리 선언표** — 내 파일들이 어디 있고 어떤 모양인지 적는 지도. 처음 세팅 때 채우고, 파일을 옮기면 여기도 같이 고칩니다.
- **내 세계 사전** — 나만 아는 판단 기준. 처음엔 거의 비어 있는 게 정상입니다.

채우는 순간은 자연스럽게 옵니다. 분류 결과를 보다가 —

> 당신: "OO보안 알림은 앞으로 항상 무시해"
> 클로드: "사전에 등록했습니다. 다음부터 자동으로 거릅니다."

이 한 마디가 영구 규칙이 됩니다. 가족 이야기가 업무 목록에 섞이는 게 싫으면 "가족 관련은 할일로 넣지 마"라고 말하면 그것도 사전에 들어갑니다.

## 뭔가 이상할 때

- **"요즘 문자가 안 들어오는 것 같은데?"** → 그대로 물어보세요. 클로드가 수집기 상태(마지막으로 언제 돌았는지)를 확인해 줍니다. 아침 상태판도 같은 걸 매일 자동으로 봅니다 — 수집이 죽으면 조용히 죽는 게 아니라 상태판에 뜹니다.
- **분류가 이상하면** → "이건 노이즈가 아니라 중요한 거야"라고 그 자리에서 교정하세요. 교정이 사전에 쌓여 다음부터 달라집니다.
- **뭐든 애매하면** → 그냥 한국어로 물어보세요. 이 시스템의 조작법은 "정확한 명령어"가 아니라 대화입니다.

## 기억할 규칙 세 줄

1. **원본은 지우지 않는다** — 수집된 것은 쌓이기만 하고, 처리 표시만 붙는다.
2. **밖으로 나가는 것은 항상 당신이 확정한다** — 발송·일정·돈은 AI가 초안까지, 마지막 한 마디는 사람.
3. **파일을 옮기면 지도(선언표)도 같이 고친다** — 안 그러면 AI가 낡은 지도를 믿는다.
