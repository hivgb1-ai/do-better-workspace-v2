## 미션 1: 채널별 매출 흐름 + "남는 장사였나"

민새롬님이 말한 그 "왜". 매출이 올랐다는 건 다 안다. 회의에서 필요한 건 **채널별로 얼마 팔렸고, 그게 남는 장사였는지**다.

> **민새롬** (디렉터): 매출 1위 채널 말고요. **마진까지요.** 무신사 잘 나간다고 좋아했는데 수수료 떼면 자사몰만 못한 거 아니냐는 얘기 지난번에 나왔잖아요.

**(속으로)**: 맞다. 매출 1등이 마진 1등이 아니다. 자사몰은 PG 수수료만, 무신사는 25%다. 같은 글루 글로스도 어디서 팔리느냐에 따라 남는 게 다르다.

**분석 요청**:
```
@sample-data/beaubble-glyf/glyf_sales.csv
@sample-data/beaubble-glyf/glyf_product_master.csv
@templates/beaubble-glyf/weekly-flow-briefing.md 양식으로
5월 GLYF 매출 흐름을 채널별로 분석해줘.
채널별 매출·비중·평균 할인율·수수료율·추정 순마진까지 한 표에 넣고,
채널마다 "왜 그렇게 움직였는지" 원인을 붙여줘.
매출 견인 SKU TOP 5랑, 마진이 새고 있는 지점도 짚어줘.
```

**결과 확인 후 추가 질문 예시**:
```
"순마진 기준으로 다시 줄 세워줘. 매출은 큰데 마진은 얇은 채널이 어디인지, 자사몰 비중을 5%p 더 올리면 전체 마진이 얼마나 좋아지는지 추정도. 추정은 가설로 표기하고"
```

## 미션 2: 매출이 튄 날, 무슨 콘텐츠가 있었나

미션 1을 9시 18분에 끝냈다. 채널 그림은 잡혔다. 한서영님 슬랙이 다시 온다.

> **한서영** (Hailey): 희경님, 중간에 체리 구가 확 튄 날 있었죠? 그거 우리 틱톡 시딩 나간 날이랑 겹치는 거 같은데, 느낌 말고 데이터로 보고 싶어요. 그래야 다음 콘텐츠를 어디에 걸지 정하죠.

**(속으로)**: 이게 내 일의 핵심이다. 매출 급등을 "단기 프로모션 / 시딩·IP 협업 / 신제품 런칭 / 정기 콘텐츠" 중 뭐가 만들었는지 분류하는 것. 느낌으로는 안다. 데이터로 보여줘야 팀이 움직인다.

**분석 요청**:
```
@sample-data/beaubble-glyf/glyf_sales.csv
@sample-data/beaubble-glyf/glyf_content_performance.csv
5월 매출 급등일(평균 대비 +30% 이상)을 추리고,
그 전후 콘텐츠랑 연결해줘.
콘텐츠를 '단기 프로모션 / 시딩·IP 협업 / 신제품 런칭 / 정기 콘텐츠' 4유형으로 나눠서,
어떤 유형이 어떤 SKU 매출을 끌었는지 정리해줘.
도달·참여는 높은데 매출로 안 간 콘텐츠도 같이 짚어줘.
```

**결과 확인 후 추가 질문 예시**:
```
"유형별 ROI로 보면 다음 주 콘텐츠는 어디에 힘을 줘야 할까? 시딩이 진짜 매출로 가는 거면, 그 매출이 틱톡샵에만 머무는지 자사몰 유입까지 가는지도 봐줘 — 인과 아니고 상관인 건 상관으로 표기하고"
```
