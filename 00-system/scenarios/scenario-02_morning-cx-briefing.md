## 미션 1: 월요일 아침 CX 한 화면 브리핑

회의 들어가기 전 한 화면. 즉시 조치 / 오늘 중 확인 / 진행 모니터링 3단계.

**(속으로)**: 정 대표님이 회의에서 가장 먼저 물어보실 것 — "오늘 가장 위험한 게 뭡니까?" 그 답이 즉시 조치 칸 맨 위에 있어야 한다. 정체일 + 심각도 기준으로 자동 정렬되면 내 직관이 빠지고 데이터가 답을 한다. 분석 템플릿에 검출 규칙(긴급+3일 정체 빨강·대책 미수립 빨강·동시/다발 batch 빨강·중대+5일 정체 노랑·VIP/관급/건설사 노랑·핵심 수치 -20% 빨강)이 다 박혀 있으니 한 번에 호출하자.

**분석 요청**:
```
@complaints.csv @directives.csv
@templates/royalco/morning-cx-briefing.md 관점으로
오늘(2026-05-13 가정) 월요일 아침 CX 브리핑 만들어줘.
즉시 조치 / 오늘 중 확인 / 진행 모니터링 3단계로.
batch 의심 건은 별도 알림 박스로 빼주고, SNS 부정 멘션 31건도 즉시 조치에 포함.
한 화면 안에 들어가야 해.
```

**결과 확인 후 추가 질문 예시**:
```
"염서우 대리 요청 — RB1530 스마트비데만 따로 추려서, 거래처·심각도·진행단계·
마지막 업데이트일로 정리해줘. 같은 batch 의심 그룹이 보이게."
```

```
"박재연 책임 SNS 신호 — 이번 주 부정 멘션 31건이 어느 제품·어느 거래처에 몰려
있는지, 사외불만 데이터에서 같은 제품 진행 중 건과 매칭해서 보여줘.
SNS 확산 위험 우선순위 3개로."
```
