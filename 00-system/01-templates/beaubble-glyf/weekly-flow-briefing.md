# GLYF 주간 매출 흐름 브리핑 양식

> 매주 월요일, 지난주 GLYF 매출이 **왜** 그렇게 움직였는지를 채널·SKU·콘텐츠 수준까지 한 번에 읽어내고, 그래서 **이번 주에 무엇을 할지**까지 정리하는 양식. 데일리 회의에서 팀과 공유하고, 확정된 액션은 노션에 옮긴다.
>
> 이 양식의 목적은 "표를 보여주는 것"이 아니라 **"왜 올랐는지 → 그래서 뭘 할지"를 답하는 것**이다. 숫자만 나열하고 끝내면 실패다. 모든 숫자 옆에는 "그래서 무슨 의미인지"가 한 줄 붙어야 한다.

## 사용 데이터

- `@sample-data/beaubble-glyf/glyf_sales.csv` — 주문 트랜잭션(채널·SKU·일자·할인) (필수)
- `@sample-data/beaubble-glyf/glyf_product_master.csv` — 채널별 수수료·원가·마진·재고 (필수, 수익성·재고 분석에 교차)
- `@sample-data/beaubble-glyf/glyf_content_performance.csv` — 콘텐츠 유형별 성과 (콘텐츠↔매출 연결 다룰 때)

매출만 보면 "얼마 팔렸나"까지밖에 못 간다. **상품 마스터와 교차해야 "남는 장사였나"**(수수료·마진), **콘텐츠 성과와 교차해야 "왜 팔렸나"**(어떤 콘텐츠가 끌었나)가 나온다. 한 파일로 끝내지 말 것.

## 출력 구조

### 1. 표지 — Editorial Cover

- 상단 풀블리드 컬러 영역(`{component.cover-band}`). 흰 텍스트.
- 라벨: `GLYF / WEEKLY FLOW BRIEFING`
- 헤드라인: **이번 주 본질 한 문장.** 단순 "매출 증가"가 아니라 원인까지. (예: "체리 구 틱톡 시딩이 주중 매출을 끌어올렸지만, 무신사 세일이 마진을 갉아먹은 한 주")
- 메타: 주차 + 데이터 기간 + 작성자(서희경) + 다음 데일리 회의 일시.

### 2. 핵심 KPI — 4-up 카드 (숫자 + 해석 1줄 필수)

각 카드는 **큰 숫자 + 전주 대비 변동 + "그래서 무슨 의미" 한 줄**로 구성. 해석 줄이 없으면 미완성.

| 카드 | 숫자 | 해석 줄 예시 |
|---|---|---|
| 주간 매출 | 7,030만원 (▲12%) | "우상향이지만 상승분의 절반이 세일 할인에서 나옴 — 질을 봐야 함" |
| 주문 건수 / 객단가 | 2,551건 / 27,600원 | "객단가 ▼3% — 세일로 단가 낮은 단품 주문이 늘어난 신호" |
| 채널 1위 | 자사몰 31.6% | "마진 가장 높은 채널이 1위 유지 — 수익 구조는 건강" |
| 매출 견인 SKU | 글루 글로스 체리 구 | "신상 아닌데 1위 — 틱톡 시딩이 만든 비정상 급등, 지속성 점검 필요" |

→ 카드 4개는 "매출 / 객단가 / 채널 구조 / 견인 SKU"가 기본. 그 주의 본질에 따라 1개는 교체 가능(예: 신제품 런칭 주에는 '신상 비중' 카드).

### 3. 채널별 매출·수익성 — 표 + 채널마다 원인 3~4줄

표: `채널 / 매출 / 비중 / 전주 대비 / 평균 할인율 / 수수료율 / 추정 순마진`

- **순마진 = (단가×(1−수수료율) − 원가) ÷ 단가.** 상품 마스터와 교차해 채널별로 계산. 이게 이 섹션의 핵심 — 매출 1위 채널이 마진 1위가 아닐 수 있다.
- 표 아래, **채널마다 3~4줄로 "왜 움직였나"**를 쓴다. 한 문장으로 끝내지 말 것:
  - 자사몰: 비중·증감 / 움직인 원인(콘텐츠·DM·재방문) / 수익성 코멘트
  - 틱톡샵: 비중·증감 / 시딩·IP 콘텐츠 영향 / 미국 시장 신호
  - 무신사·ZALORA(동남아): 비중·증감 / 세일·해외 런칭 영향 / **수수료로 인한 마진 누수 경고**
- 강조 1행: 그 주 가장 크게 움직인 채널(`{colors.accent-soft}` 배경).

### 4. SKU 분석 — TOP + 공통점 + 하위 위험군

- **TOP 5 표**: SKU / 상품명 / 카테고리 / 판매수량 / 매출 / 추정 마진율.
- **TOP의 공통점 분석 2~3줄**: 색상 경향(코랄·피치 등 웜톤이 끄는지), 가격대, 신상 여부, 어떤 채널에서 주로 팔렸는지. "왜 이 5개가 위로 왔나"를 해석.
- **하위·위험 SKU 신호**:
  - 재고 임박(아래 6번과 연결): 잘 나가는데 재고 부족한 SKU.
  - 부진 SKU: 라인업에 있는데 거의 안 팔리는 SKU — 단종·리뉴얼·번들 검토 후보.
  - 과재고: 안 나가는데 재고만 쌓인 SKU(예: 라이너 계열).

### 5. 콘텐츠 ↔ 매출 연결 (콘텐츠 성과 교차 시)

서희경 업무의 핵심. **"매출 급등일 = 어떤 콘텐츠가 있었나"를 콘텐츠 유형 4종으로 분류해 연결한다.**

- 콘텐츠 유형: **단기 프로모션 / 시딩·IP 협업 / 신제품 런칭 / 정기 콘텐츠**.
- 매출 급등일(전일 대비 또는 평균 대비 +30%↑)을 추려, 그 1~2일 전후의 콘텐츠와 매칭.
- 유형별로 정리: 어떤 유형이 어떤 SKU 매출을 얼마나 끌었나 / 도달·참여 대비 실제 유입클릭·매출 전환(ROI 상하위).
- **결론 줄**: "다음 주 콘텐츠는 어떤 유형에 힘을 줄지" 1~2줄. (이게 6번 액션의 콘텐츠 항목으로 이어짐)

### 6. 재고·주의 신호 — 3단 분류 + 액션

매출이 올라도 재고가 못 받치면 기회 손실. 상품 마스터의 재고·월평균 판매로 소진 속도를 계산해 3단으로 분류하고, **각 단에 액션과 기한**을 붙인다.

- 🔴 **긴급** (소진 2주 미만): 잘 나가는데 곧 품절. → 재발주 결정 기한 명시(예: "6/4까지 발주 결정").
- 🟡 **주의** (2~4주): 모니터링. → 다음 주 판매 보고 재판단.
- 🟢 **정상** (4주+): 별도 액션 없음. (정상은 1줄로 압축, 길게 쓰지 말 것)
- 📦 **과재고**: 소진 12주+ → 번들·세일·콘텐츠 노출 등 소진 전략 제안.

### 7. 회의용 코멘트 후보 — 3종 (비워두지 말 것)

데일리 회의에서 서희경이 한 문장으로 흐름을 말할 때 쓸 후보를 **미리 3가지 톤으로** 뽑아둔다. 비워두고 "직접 채우세요"로 끝내지 말 것 — 후보가 있어야 고른다.

1. **숫자 중심**: "지난주 매출 ▲12%, 그중 절반이 세일 할인에서 나와 순마진은 보합. 견인은 체리 구 틱톡 시딩."
2. **흐름 중심**: "틱톡 시딩이 통한다는 게 두 번째로 확인됐어요. 다음은 이걸 자사몰 유입으로 어떻게 넘기느냐예요."
3. **헤드라인 톤**: "잘 판 한 주가 아니라, 잘 판 '이유'를 안 한 주."

### 8. 이번 주 액션 — 담당 / 액션 / 기한 / 우선순위 (4열 표)

"다음 주 액션 3줄"로 뭉개지 말 것. **구조화된 표 + 성공 기준**으로.

| 담당 영역 | 액션 | 기한 | 우선순위 |
|---|---|---|---|
| 신제품 런칭 | 하이라이터 팔레트 재발주 GO/NO 결정 | 6/4 | 🔴 |
| 시딩·IP | 체리 구 시딩 2차 — 자사몰 유입 링크 포함 설계 | 6/6 | 🟡 |
| 콘텐츠 | 다음 주 정기 콘텐츠를 웜톤 립으로 집중 | 6/5 | 🟡 |

- 액션은 **5번(콘텐츠 방향)·6번(재고)에서 도출**된 것이어야 함. 새로 지어내지 말 것.
- 각 액션의 담당 영역은 서희경 업무 기준(IP 리드 / 시딩 / 신제품 런칭 / 콘텐츠)으로 태그.
- 가능하면 "성공 기준"(무엇이 충족되면 된 것인지) 한 줄 덧붙임.

## 검출 규칙 (Anomaly Thresholds)

- **매출 급등일**: 전일 대비 또는 기간 평균 대비 +30% 이상 → 5번에서 콘텐츠 매칭 대상.
- **마진 누수**: 채널 평균 할인율 > 0.18 또는 (수수료율 + 평균 할인율) > 0.40 → 3번에서 "마진 누수" 경고.
- **재고 긴급**: 재고 ÷ 주간 판매속도 < 2주 → 6번 🔴.
- **과재고**: 재고 ÷ 주간 판매속도 > 12주 → 6번 📦.
- **부진 SKU**: 주간 판매량이 라인 평균의 30% 미만 → 4번 하위 위험군.
- **콘텐츠 고ROI**: 유입클릭 대비 해당 SKU 매출 전환 상위 → 5번에서 "다음 주 강화" 후보.

## 길이 제약

- A4 1~2페이지. 데이터 더 들어가면 페이지 추가, 표 압축 금지.
- 본문 문단은 3~5줄 이내. 단, **2·3·5번의 "해석/원인" 줄은 생략 금지** — 이게 이 양식의 핵심이다.
- 차트는 1~2개(주차별 매출 추이 + 채널 비중 정도). 더 필요하면 별첨.

## 톤 가이드

- 팀 공유용. Co-founder가 팀에게 "그래서 이렇게 하자"를 말하는 톤. 결론·근거 순서.
- **모든 숫자 옆에 "그래서 무슨 의미"**가 따라붙어야 한다. 숫자만 있는 줄은 미완성으로 본다.
- 추측·가설에는 "(추정)" / "(가설)" 표기. 콘텐츠↔매출 인과는 상관일 뿐이므로 "(상관)" 명시.
- 특정 팀원을 탓하는 표현 금지. 원인은 채널·콘텐츠·시즌 등 구조로 설명.
- 자기 PR 금지("잘했다" 류 없음).
