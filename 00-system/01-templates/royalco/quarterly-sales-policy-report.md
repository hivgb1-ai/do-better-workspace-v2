# 분기 영업정책 이행 현황 보고서 템플릿

> 로얄앤컴퍼니 영업총괄이 분기마다 임원진(정 대표 포함)에게 올리는 1~2페이지 보고서.
> 매 분기 같은 양식으로 나오게 박아두는 용도.

## 입력 데이터

| 파일 | 역할 |
|---|---|
| `sales_weekly.csv` | 영업사원·거래처별 주차별 매출 (14주분) |
| `customers.csv` | 거래처 마스터 (채널·약정·환원유예 보유 여부) |
| `special_requests.csv` | 특별출고·특별가 요청 (최근 30일) |
| `receivables.csv` | 매출채권 잔액 + 연체 구간 + 이행계획 대비 실적 |

## 분석 윈도

- **이번 분기**: 보고 시점 기준 직전 13주
- **비교 기준**: 직전 분기 (지지난 13주, 데이터에 없으면 가용 범위)
- 데이터 마지막 주차를 "이번 분기 마감"으로 간주.

## 보고서 출력 구조 (5섹션, A4 1~2장)

### 1. 한 줄 요약 (헤더)
```
[YYYY-MM-DD ~ YYYY-MM-DD] 분기 매출 X.XX억 / 약정 대비 N% / 의사결정 안건 K건
```

### 2. 채널별 약정 이행 표 (5행)
컬럼: 채널 / 분기 매출 / 분기 약정 / 이행률 / 거래처 수 / 약정 미달 거래처 수 / 알림
- 채널 5개 전부 (전문점·대리점·건설사·직영점·수출)
- 정렬: 매출 내림차순

### 3. 영업사원별 실적 표 (영업사원 10명)
컬럼: 영업사원 / 분기 매출 / 약정 합계 / 이행률 / 환원유예 보유 거래처 수 / 알림
- 정렬: 이행률 오름차순 (미달자 먼저 보이게)

### 4. 특별요청 검토 알림 (최근 30일)
컬럼: 요청일 / 거래처 / 유형 / 사유 / 원가배수(특별가) or 수량(특별출고) / 전월 보고 이행 / 상태 / 판정
- 원가배수 < 1.45 또는 전월 보고 미이행 또는 검토중 상태 → 임원 확인 필요로 표시
- 5건 넘으면 임팩트 큰 5건만

### 5. 의사결정 후보 (3개 이내)
형식: 각 항목마다 (배경 한 줄 / 데이터 근거 한 줄 / 선택지 2~3개 / 임원진 의견 자리 빈 칸)
- 발견된 신호 중 분기 회의 안건이 될 만한 것
- 3개를 넘으면 임팩트 큰 3개만 (매출 임팩트 또는 매출채권 임팩트 기준)

## 검출 규칙 (이상 신호)

| 신호 | 기준 | 색상 |
|---|---|---|
| 채널 이행률 미달 | 분기 약정 대비 < 85% | 노랑 |
| 채널 이행률 심각 미달 | 분기 약정 대비 < 70% | 빨강 |
| 영업사원 이행률 미달 | 분기 약정 대비 < 80% | 노랑 |
| 거래처 매출 급락 | 최근 4주 vs 직전 9주 평균 -35% 이하 | 빨강 |
| 거래처 매출 급등 | 최근 4주 vs 직전 9주 평균 +45% 이상 | 초록 (확장 후보) |
| 특별가 원가배수 미달 | 원가배수 < 1.45 | 빨강 |
| 전월 보고 미이행 거래처 특별요청 | prev_month_report_ok = N | 노랑 |
| 매출채권 악성 잔액 | 90일 초과 > 0 | 빨강 |
| 환원유예 보유 거래처 매출 미달 | has_deferred_quota = Y + 이행률 < 80% | 빨강 |
| 매출채권 회수율 미달 | actual / plan_target < 75% | 노랑 |

## 길이·톤 제약

- **A4 1~2장 안**. PDF 변환 시 표 위주 (이미지 0개).
- 표는 마크다운 표. 셀 안에 줄바꿈 사용 금지.
- 임원이 5분 안에 핵심 파악 가능하게. 본문 산문 최소화, 표·알림 위주.
- 톤: 사실·숫자 중심. 추정은 "추정" 명시.
- 데이터에 없는 외부 추정 금지. 모르면 "데이터 없음" 명시.
- 환원유예 케이스는 별도 분리 표시 — 일반 미달과 섞지 않는다.

## 산출물 형식

기본: 마크다운(.md). 임원 검토 후 한 줄 요청으로 PDF용 HTML(로얄앤코 디자인) 변환.

## 주의

- **조인호 부장 본인이 보고 직전에 추가 질문을 던지는 흐름을 가정한다.** 보고서 1차 산출 후 "이 영업사원 왜 떨어진 거야?" 같은 즉문에 동일 데이터 안에서 답할 수 있어야 한다 (외부 추정 금지).
- 환원유예 거래처는 일반 미달 거래처와 분리 표시 — 임원이 한눈에 구분 가능해야 함.
- 거래처 식별자는 customer_id 기준. 거래처명이 비슷한 경우 혼동 방지.
- 분기 약정 = 연간 약정 / 4 (만원 단위).
