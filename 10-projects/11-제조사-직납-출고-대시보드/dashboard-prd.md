# 제조사 직납 출고 대시보드 PRD

> 생성일: 2026-07-08
> 생성 방법: dashboard-prd skill (대화형)

## 1. 프로젝트 개요

### 배경 (Pain Points)
- 제조사별·품목별 월간 직납 출고량과 전월 대비 비중 변화를 확인할 별도 체계가 없어, 지금까지 이 비교를 해본 적이 없음
- 비용절감 계산이 매달 수동으로 새로 만들어지는 엑셀 파일(`_06월.xlsx`, `_07월.xlsx` ...)로 관리되어, 출고 데이터와 분리되어 있고 히스토리 추적이 번거로움

### 핵심 의사결정
- 월별/제조사별로 어떤 품목이 얼마나 출고되었는지 파악
- 전월 대비 어떤 품목의 직납 출고 비중이 오르고 내렸는지 파악
- 절감비율(절감액 ÷ 매출액)이 목표(1.6%) 대비 어떤 진행 상태인지 파악

### 목적
쿠팡 제조사 직납 출고 데이터를 월별/기간별로 한눈에 파악하여, 품목별 비중 변화와 비용절감 목표 달성 여부를 신속히 확인한다.

### 핵심 가치
1. 매번 시트를 열어 수작업으로 비교하지 않아도 전월 대비 변화가 바로 보임
2. 3/6개월 등 기간을 선택하면 그 기간의 누적 증감을 바로 확인
3. 출고 데이터와 비용절감 계산이 한 곳에서 연결되어 관리됨
4. 데이터 오류(#REF!, 미출고 등)가 조용히 집계에서 제외되어 숫자 신뢰도 유지

## 2. 데이터 소스

### 소스 1: 음료 제조사 직납 출고내역 (Google Sheets)
- **위치**: https://docs.google.com/spreadsheets/d/1kM2bt4cvpERfZgmLcs_HCe4usgxQdvcx5o8_cZLV9n0 (탭: "음료 제조사 직납 출고내역")
- **형태**: Google Sheets 탭, 일자별 트랜잭션 로그 (연속 히스토리, 연도는 "2026년도 출고내역" 같은 섹션 헤더로 구분)
- **주요 컬럼**: 매입(ERP 매입 처리 완료 여부, 집계 무관), 날짜(M/D), 요일, 보내는창고(제조사: 튤립·오케이에프·건강한사람들 등), 받는창고(쿠팡), ERP 코드, 상품명, 입수량, 요청 수량(EA), BOX 수량, 소비기한, 쿠팡 입고 센터
- **특이사항**:
  - 일부 행에 `#REF!` 오류 (수식 깨짐 — ERP코드/상품명/입수량/요청수량/BOX수량 전체 손상)
  - "요청 수량" 컬럼에 숫자 대신 "미출고" 텍스트가 들어간 행 존재
  - 날짜에 연도가 없어 섹션 헤더 기반 연도 파싱 필요
- **정제 규칙**: `#REF!` 오류 행과 "미출고" 텍스트 행은 집계에서 완전히 제외하고 별도 표시도 하지 않음 (사용자 확정)

### 소스 2: 애사비소다 제조사 직납 비용절감 (현재: 월별 Excel → 추후: Google Sheets 이관)
- **현재 위치**: `G:\공유 드라이브\HQ_Team_SCM\SCM_공용\★성과측정지표(KPI)\2026년\물류운영\쿠팡 비용 절감\제조사 직납\애사비소다 제조사 직납 비용절감_MM월.xlsx` (매월 새 파일 생성)
- **형태**: Excel, 2개 시트
  - **원가산출V2** (67행×98열): 제조사×품목별 행, 월별(1~12월) 출고량(EA/PLT) 입력 → 단가(보관비 700원×4일, 운송비 원가반영, 작업비 EA 5원/PLT 15,000원)를 곱해 절감액(보관비/운송비/작업비/TTL) 자동 계산. 다단 헤더(제조사/품목/단가/월별 출고량·절감액 서브컬럼) + 상단 합계 메타데이터 행
  - **직납비율**: 월별 "직납" 매출액 vs "밀크런&쉽먼트" 매출액(애사비소다 품목군 매출 중 채널별 비중), TTL, 채널별 비중(%), 상반기/하반기/YTD 합계
- **결정 사항**: 이 비용 계산 구조를 Google Sheets로 이관하여 출고내역 시트와 동일한 방식으로 연동 관리 (별도 이관 작업 필요 — §3 데이터 파이프라인 참고)
- **핵심 KPI 산식**: 절감비율(%) = 절감액 TTL(원가산출V2, 해당 월) ÷ 매출액 TTL(직납비율, 해당 월) × 100, **목표 1.6%**

## 3. 데이터 파이프라인

### 전체 흐름
```
[음료 제조사 직납 출고내역]          [애사비소다 비용절감 (Google Sheets 이관 후)]
   Google Sheets 탭                     원가산출V2 탭          직납비율 탭
        |                                    |                     |
        | (정제: #REF!/미출고 행 제외)         | (제조사x품목x월별       | (월별 매출액
        | (연도 파싱: 섹션 헤더 기반)           |  절감액 계산)          |  채널 비중)
        v                                    v                     v
   ┌─────────────────────────────────────────────────────────────────┐
   │        통합 뷰: 월별 x 제조사 x 품목 출고량 + 절감비율              │
   └─────────────────────────────────────────────────────────────────┘
        |
        v
   Next.js 대시보드 (ISR 캐싱, ★기간 필터: 당월/전월/3개월/6개월/기간지정)
```

### 선행 작업 (구현 전 필요)
1. **비용절감 엑셀 → Google Sheets 이관**: 원가산출V2의 다단 헤더/수식 구조를 Google Sheets 수식(또는 Apps Script)으로 재현. 매달 새 파일을 만드는 방식 대신 연속 시트로 전환
2. **날짜 정규화 로직 확정**: 출고내역 시트의 "M/D + 연도 섹션 헤더" 구조를 실제 날짜(YYYY-MM-DD)로 변환하는 규칙 확정(섹션 헤더 파싱)

### 통합 테이블 스키마 (제안)
| 컬럼명 | 타입 | 소스 | 설명 |
|--------|------|------|------|
| date | date | 출고내역 | 정규화된 출고일 |
| manufacturer | string | 출고내역 (보내는창고) | 튤립/오케이에프/건강한사람들 등 |
| erp_code | string | 출고내역 | 품목 식별자 |
| product_name | string | 출고내역 | 상품명 |
| qty_ea | number | 출고내역 (요청 수량) | 출고 수량 |
| savings_total | number | 비용절감(원가산출V2) | 해당 월 제조사×품목 절감액 TTL |
| revenue_direct | number | 비용절감(직납비율) | 해당 월 직납 채널 매출액 |
| revenue_milkrun | number | 비용절감(직납비율) | 해당 월 밀크런&쉽먼트 채널 매출액 |
| is_excluded | boolean | 파생 | #REF!/미출고 여부 (true면 집계 제외) |

## 4. 대시보드 화면 (기능 정의 — 시각 위계는 디자인 툴 위임)

### View 1: 이번 달 개요 — 이번 달 출고 현황과 전월 대비 변화를 한눈에 파악해 이상 징후(비중 급변 품목, 절감비율 목표 미달)를 포착한다

**KPI 항목** (우선순위 순):
| 순위 | 지표명 | 비교 기준 | 출처 | 계산 근거 |
|------|--------|-----------|------|-----------|
| 1 | 절감비율 | 목표 1.6% | 비용절감(원가산출V2 + 직납비율) | 절감액TTL ÷ 매출액TTL × 100 |
| 2 | 총 출고량 | 전월 대비 | 출고내역 | 정제 후 요청수량(EA) 합계 |
| 3 | 제조사 비중 하이라이트 | 전월 대비 | 출고내역 | 제조사별 출고량 비중 최대/최소 변화 |
| 4 | 집계 제외 건수 | — | 출고내역 | #REF!/미출고로 제외된 행 수 |

**표시하고 싶은 데이터 관계** (우선순위 순):
| 순위 | 데이터 관계 | 출처 |
|------|-------------|------|
| 1 | 월별 제조사별 출고량 추이를 제조사별로 비교 (최근 6개월) | 출고내역 |
| 2 | 월별 절감비율 추이를 목표선(1.6%)과 함께 비교 | 비용절감 |
| 3 | 전월 대비 비중이 가장 많이 오르고 내린 품목 TOP N | 출고내역 |
| 4 | 품목×제조사 교차 출고량(당월) | 출고내역 |

**필터**: 기간(당월/전월/3개월/6개월/기간 지정), 제조사, 품목

**데이터 투명성**: 집계 제외 건수(카드)로만 노출, 제외된 개별 행 내역은 별도 표시하지 않음 (사용자 확정). 연도 미표기 날짜는 섹션 헤더 기반 정규화 사실을 데이터 소스 노트에 명시

**AI 인사이트 제안 패널**:
| 유형 | 탐지 조건 | 예시 문구 |
|------|-----------|-----------|
| 이상 탐지 | 품목 비중 전월 대비 ±5%p 이상 변화 | "클룹 애사비소다 오리지널 500ml 비중 +7.4%p (튤립)" |
| 추세 경고 | 절감비율이 2개월 연속 목표(1.6%) 미달 | "절감비율 2개월 연속 목표 미달 (1.2%→0.9%)" |
| 기회 발견 | 특정 제조사 출고 비중 급증 + 절감비율 개선 동반 | "튤립 출고 비중 확대와 함께 절감비율 상승 추세" |

### View 2: 기간별 히스토리 조회 — 선택한 기간(3개월/6개월/커스텀) 동안 출고량과 절감비율이 얼마나 증감했는지 파악해 다음 분기 전략에 반영한다

**KPI 항목** (우선순위 순):
| 순위 | 지표명 | 비교 기준 | 출처 | 계산 근거 |
|------|--------|-----------|------|-----------|
| 1 | 선택 기간 누적 출고량 증감 | 직전 동일 길이 기간 | 출고내역 | 기간 합계 비교 |
| 2 | 선택 기간 평균 절감비율 | 목표 1.6% | 비용절감 | 기간 내 월별 절감비율 평균 |
| 3 | 기간 내 비중 변화 최대 품목 | 기간 시작 대비 종료 | 출고내역 | 비중 변화폭 순위 |

**표시하고 싶은 데이터 관계** (우선순위 순):
| 순위 | 데이터 관계 | 출처 |
|------|-------------|------|
| 1 | 선택 기간 동안의 월별 출고량 추이를 제조사별로 비교 | 출고내역 |
| 2 | 선택 기간 동안의 절감비율 추이를 목표선과 비교 | 비용절감 |
| 3 | 기간 시작 대비 종료 시점 품목별 비중 변화 순위 | 출고내역 |

**필터**: 기간 프리셋(3개월/6개월) + 커스텀 범위(연/월 선택), 제조사, 품목

**데이터 투명성**: 기간 비교는 "직전 동일 길이 기간" 기준임을 명시 (예: 최근 3개월 선택 시 그 직전 3개월과 비교)

## 5. 기술 스택

| 기술 | 용도 | 선택 이유 |
|------|------|-----------|
| Next.js | 프레임워크 | ISR로 시트 갱신을 준실시간 반영 |
| Google Sheets (gws CLI 또는 Sheets API) | 데이터 레이어 | 출고내역·비용절감 모두 사용자가 직접 시트에서 수정 가능. 워크숍 표준 |
| shadcn/ui | UI 컴포넌트 | 기본 테마 채택 |
| Recharts | 차트 | 컬럼 차트, 라인(목표선), 진행률 링 구현 |
| Tailwind CSS | 스타일링 | 유틸리티 기반 |
| TypeScript | 타입 안전성 | — |

> **주의**: 현재 이 워크스페이스 환경에는 `gws` CLI가 설치되어 있지 않음. 구현 착수 전 아래 중 하나가 필요:
> 1. `gws` CLI 설치 및 Google 계정 연동, 또는
> 2. Google Cloud 서비스 계정 발급 → Sheets API 활성화 → 대상 시트에 서비스 계정 이메일을 조회 권한으로 공유
>
> 비용절감 엑셀(Google 공유 드라이브에 위치)의 Google Sheets 이관 작업도 구현 착수 전 선행되어야 함 (§3 참고)
